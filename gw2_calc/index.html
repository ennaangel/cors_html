Calculates cost of Agony Resist

This small Javascript experiment makes use of the < href="https://wiki.guildwars2.com/wiki/API:Main"> Guildwars2 API to pull about the cost of a single item(infusion) and displays it. Subsequently, it calculates the total price to create a +9 infusion from the current infusion item using the basic equation seen below.
Total Cost = (Cost of a single infusion)*(Number of infusions needed) + Base cost of upgrading
